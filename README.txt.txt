# Proyecto de Noticias con React

Este proyecto es una aplicación web de noticias que utiliza React para el frontend y una API de noticias para obtener y mostrar los titulares más recientes en la categoría de negocios.

## Uso

Para ejecutar la aplicación en modo de desarrollo, utiliza el siguiente comando:

```bash
npm run dev

Esto le mostrara el link de manera local al cual deberá entrar con el presionando la tecla Ctrl + Click derecho.
Posteriormente podrá entrar a su navegador y en la barra superior se mostrara el botón Api, el cual mostrara las noticias.
